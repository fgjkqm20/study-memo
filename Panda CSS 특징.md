# Panda CSS 특징

제로 런타임 CSS-in-JS 엔진

**정적 분석**: 빌드 시점에 스타일 구문 분석 후 CSS 파일을 생성

**PostCSS**: 빌드 시 파싱된 데이터를 원자적 CSS로 변환

**Codegen**: 스타일을 작성하는 데 사용되는 가벼운 런타임 JS 코드 생성

**유형 안전성**: csstypeCSS 속성과 디자인 토큰에 대한 유형 안전성을 제공

**성능**: 최적화된 원자적 CSS 파일을 생성하는 고유한 접근 방식을 사용

**개발자 경험**: 레시피, 패턴, 디자인 토큰, JSX 스타일 속성 등 풍부한 기능 세트

**최신 CSS 기능**: 캐스케이드 레이어, CSS 변수, :where, :is 같은 최신 선택자 등
